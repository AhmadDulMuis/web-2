1.Jelaskan apa yang di maksud dengan dokumen root?
Yaitu adalah direktori utama di mana server web mencari file yang akan disampaikan kepada pengguna, ini adalah lokasi di mana file HTML, PHP, CSS, JavaScript, dan aset web lainnya disimpan untuk diakses melalui web browser.
2.Pada Apache web server, apakah dokumen root harus terletak dalam direktori C:\xampp\htdocs? Jelaskan jawaban Anda.
Tidak, dokumen root tidak harus selalu terletak dalam direktori C:\xampp\htdocs. Meskipun ini adalah lokasi default untuk XAMPP, pengguna dapat mengubah dokumen root dengan mengedit file konfigurasi Apache (httpd.conf). Pengaturan dapat diubah dengan memodifikasi direktif DocumentRoot dan Directory untuk menunjuk ke direktori lain.
3.Jelaskan apa yang dimaksud dengan perintah git berikut:
init: Menginisialisasi repository Git baru di dalam direktori proyek.
clone: Menyalin repository Git yang sudah ada dari remote ke lokal.
remote: Mengelola daftar remote repository yang terhubung dengan repository lokal.
add: Menambahkan perubahan file ke staging area sebelum dilakukan commit.
commit: Menyimpan perubahan yang telah ditambahkan ke staging area ke dalam repository.
push: Mengunggah perubahan dari repository lokal ke repository remote.
checkout: Berpindah antar branch atau mengembalikan file ke kondisi sebelumnya.
4.Jelaskan perbedaan antara GitHub dan GitLab.
GitHub dan GitLab adalah platform berbasis Git untuk manajemen kode versi, namun memiliki beberapa perbedaan:
GitHub lebih banyak digunakan oleh komunitas open source dan memiliki fitur integrasi dengan berbagai layanan pihak ketiga.
GitLab menawarkan lebih banyak fitur built-in, termasuk CI/CD tanpa memerlukan layanan tambahan.
GitLab memungkinkan pengguna untuk meng-host repository mereka sendiri di server pribadi dengan edisi self-hosted-nya.
5.Apa bedanya variable user dengan variable system?
Variable User adalah variabel lingkungan yang hanya berlaku untuk pengguna tertentu dalam sistem operasi.
Sedangkan Variable System adalah variabel lingkungan yang berlaku untuk semua pengguna dalam sistem operasi.
6.Apa persamaan variable system dengan variable constant?
Persamaannya yaitu keduanya digunakan untuk menyimpan nilai yang digunakan oleh sistem atau program.
Nilai dari variable system dan variable constant biasanya tidak berubah selama eksekusi program atau sesi sistem berjalan.
Keduanya digunakan untuk mengatur lingkungan eksekusi program atau sistem operasi.
